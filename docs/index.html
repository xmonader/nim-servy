nim-servy

A lightweight async web framework for Nim

GitHub Docs Nim 2.x
Features Install Quickstart Routing Middleware WebSocket

Features

Async First

Built on Nim's asyncdispatch. All handlers and middleware are async by default.

Zero Dependencies

Only depends on terminaltables. WebSocket support built from scratch.

Route Parameters

Capture URL segments with :param syntax. Access via req.urlParams.

Form Data

Parse URL-encoded and multipart form bodies automatically.

Middleware

Per-route or global middleware. Built-in logging, static files, and basic auth.

WebSocket

Full WebSocket support with handshake, framing, and echo. No external deps.

Installation

nimble install servy

Quickstart

import servy

when isMainModule:
  var router = initRouter()

  proc handleHello(req: Request, res: Response): Future[void] {.async.} =
    res.code = Http200
    res.content = "hello world!"

  router.addRoute("/hello", handleHello)

  let opts = ServerOptions(address: "127.0.0.1", port: 9000.Port)
  var s = initServy(opts, router)
  s.run()

Visit http://localhost:9000/hello

Routing

Basic routes

router.addRoute("/", handleIndex)
router.addRoute("/get", handleGet, HttpGet)
router.addRoute("/post", handlePost, HttpPost)

URL parameters

# /user/:user captures the segment into req.urlParams["user"]
router.addRoute("/user/:user", handleUser)
router.addRoute("/multi/:first/:second", handleMultiParam)

Query parameters

# GET /query?page=1&limit=10
# req.queryParams["page"] == "1"
router.addRoute("/query", handleQuery)

Form data

# POST with application/x-www-form-urlencoded
proc handleForm(req: Request, res: Response): Future[void] {.async.} =
  let user = req.formData.getValueOrNone("username")
  res.content = "user=" & user.get("")

router.addRoute("/login", handleForm, HttpPost)

Middleware

Middleware procs return Future[bool]. Return true to continue, false to short-circuit.

Per-route middleware

let users = {"admin": "secret"}.toTable
router.addRoute("/admin", handleAdmin, HttpGet, @[
  basicAuth(users)
])

Global middleware

var s = initServy(opts, router, @[
  loggingMiddleware,
  trimTrailingSlash,
  newStaticMiddleware("public", "/static")
])

Built-in middleware

loggingMiddleware

Logs request method and path.

trimTrailingSlash

Strips trailing slashes from request path.

newStaticMiddleware

Serves static files from a directory.

basicAuth

HTTP Basic Authentication with user/password table.

WebSocket

Built-in WebSocket support with handshake, framing, and ping/pong handling.

proc handleWS(req: Request, res: Response): Future[void] {.async.} =
  var ws = await newServyWebSocket(req)
  await ws.send("Welcome!")
  while ws.readyState == Open:
    let packet = await ws.receiveStrPacket()
    await ws.send("echo: " & packet)

router.addRoute("/ws", handleWS, HttpGet)
